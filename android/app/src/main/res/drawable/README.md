# Android Drawable Resources para Native Bottom Tabs

Este diretório contém os ícones nativos do Android usados nas Bottom Tabs.

## Ícones Necessários

Para que as Native Bottom Tabs funcionem corretamente no Android, você precisa adicionar os seguintes arquivos XML neste diretório:

### 1. ic_moments_filled.xml
```xml
<vector xmlns:android="http://schemas.android.com/apk/res/android"
    android:width="24dp"
    android:height="24dp"
    android:viewportWidth="24"
    android:viewportHeight="24">
    <path
        android:fillColor="@android:color/white"
        android:pathData="M12,2C6.48,2 2,6.48 2,12s4.48,10 10,10 10,-4.48 10,-10S17.52,2 12,2zM10,16.5v-9l6,4.5 -6,4.5z"/>
</vector>
```
